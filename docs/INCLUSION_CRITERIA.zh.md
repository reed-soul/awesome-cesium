# Awesome Cesium - 收录标准

本文档定义 Awesome Cesium 列表的正式收录标准。所有提交的资源均需符合以下标准。

## 核心原则

1. **质量优于数量** - 仅收录维护良好、切实有用的资源
2. **直接相关性** - 必须扩展、增强或集成 CesiumJS
3. **准确性** - 所有链接必须有效，描述必须准确

---

## 收录要求

### 所有资源

| 标准 | 要求 |
|------|------|
| **Cesium 关联** | 与 CesiumJS、3D Tiles 或 Cesium 生态直接相关 |
| **文档** | 具有清晰的 README、文档或使用示例 |
| **唯一性** | 不重复已有资源 |

### GitHub 仓库

| 标准 | 要求 | 例外 |
|------|------|------|
| **最后提交** | 24 个月内有提交 | 官方/历史意义项目 |
| **README** | 有完整的 README | - |
| **许可证** | 有明确的许可证信息 | - |
| **可用性** | 能按描述正常使用 | - |

### 文档/教程

| 标准 | 要求 |
|------|------|
| **准确性** | 内容当前、正确 |
| **清晰度** | 结构清晰、易读 |
| **示例** | 包含实用示例 |

### 工具/应用

| 标准 | 要求 |
|------|------|
| **稳定性** | 功能正常、可靠 |
| **安装** | 有清晰的安装说明 |
| **价值** | 解决 Cesium 开发者的实际需求 |

---

## 排除标准

以下资源**不会**被收录：

- 已弃用（24 个月以上无提交，除非具有官方/历史意义）
- 链接失效或无法访问
- 与维护更好的替代品功能重复
- 与 Cesium 无明确关联
- 缺少文档或使用示例
- 仅商业、无免费/开源组件的项目

---

## 特殊标签

| 标签 | 使用场景 |
|------|----------|
| `⚠️ **[已过时 - 年份]**` | 24 个月以上无提交，保留供参考 |
| `🧪 **[实验性]**` | 不稳定或前沿项目 |
| `🆕 **[新]**` | 最近 3 个月内添加 |
| `🌟 **[热门]**` | 广泛采用（高 star/使用量） |

---

## CesiumJS 版本兼容性

- **当前 CesiumJS**：1.x（参见 [ releases ](https://github.com/CesiumGS/cesium/releases)）
- 添加资源时，如已知兼容性请注明
- 标记为已过时的资源可能针对较旧版本 Cesium

---

## 维护

- **链接检查**：每周（自动化）
- **质量审计**：每月（自动化，检查仓库活跃度）
- **人工审核**：每季度检查分类结构
- **标准更新**：随 Cesium 生态发展而更新
